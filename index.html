🥞 Call Stack: Let's Experiment Together

In this interactive demo, we are going to explore how the Call Stack works in JavaScript. By clicking the "Run Demo" button, we will see how functions are called and managed in the call stack.

Watch the console and the Call Stack visualization as the functions execute.

Code we are going to run:

function cook() {
    console.log("🍳 1. Starting to cook...");
    washDishes();
    console.log("🍳 6. Finished cooking!");
}

function washDishes() {
    console.log("🧽 2. Starting to wash dishes...");
    dryDishes();
    console.log("🧽 5. Finished washing dishes!");
}

function dryDishes() {
    console.log("🏠 3. Starting to dry dishes...");
    putAwayDishes();
    console.log("🏠 4. Finished drying dishes!");
}

function putAwayDishes() {
    console.log("📦 Putting dishes in the cupboard");
}
